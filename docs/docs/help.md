# Help / Get Help

## Do you want to help us?

Pull requests are always welcome.

You can inspect our docs for typos and spelling mistakes, and create pull requests or [open an issue](https://github.com/0xff8c00/django-hattori/issues).

If you have any suggestions to improve **Django Hattori**, please create them as [issues](https://github.com/0xff8c00/django-hattori/issues) on GitHub.


## Do you need help?

Do not hesitate.  Go to [GitHub issues](https://github.com/0xff8c00/django-hattori/issues) and describe your question or problem.  We'll attempt to address them quickly.
